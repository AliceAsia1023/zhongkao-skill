# 深圳中考志愿填报助手 · 网页工具

面向家长的可视化志愿填报工具，基于本 skill 的决策框架构建。

## 功能

- 输入预估分数、初中名称、住宿偏好、区域偏好、性格特点
- 自动分析分数档位（顶尖/优质/良好/普通等）
- 生成名额分配（指标生）志愿建议
- 生成12个统招志愿完整布局（彩票/超常/稳定/兜底四层）
- 内置个性化注意事项（走读调剂警告、分制换算提醒等）

## 学校数据库

- 来源：梧桐论坛深户AC类历年录取分数（2021-2025年）
- 覆盖学校：**102所**（含公办A+/A/B+/B 及部分民办）
- 分数范围：354 ~ 593分（2026年630分制估算）
- 91所学校有2025年实录数据，11所使用2024年估算

> ⚠️ 2026年满分从610→630，所有历史分数已按比例换算（×630/610），
> 具体以当年《考生报考指导手册》为准。

## 部署

静态 HTML，无需后端，任意 Web 服务器可直接托管：

```bash
# nginx
cp index.html /var/www/html/zhongkao/

# Python 快速预览
python3 -m http.server 8080
```
